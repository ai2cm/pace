pace-physics¶

pace-physics includes the Python implementation of GFS physics built using the GT4Py domain-specific language. Currently, only GFDL cloud microphysics is integrated into Pace. Additional physics schemes (NOAH land surface, GFS sea ice, scale-aware mass-flux shallow convection, hybrid eddy-diffusivity mass-flux PBL and free atmospheric turbulence, and rapid radiative transfer model) have been ported indendepently and are available in the physics-standalone repository. Additional work is required to integrate these schemes.
